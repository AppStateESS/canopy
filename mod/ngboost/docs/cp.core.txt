ControlPanel ngBoost Tab "Core"

Core modules are installed during setup of the site and are not intented to uninstall.
Click "Check" or "CheckAll" to verify, if new versions are available.

With "CheckAll", core and application modules are verified.

The link "About" tells some infos about the module.

The link "Backup" produces a backup of the module from the file system of the server.
Note, to have backups from the database, use the similar action on tab "db".
Backed up data are placed in a repository. To have an overview of the backups just
made, walk to tab "repository" and click the link "ListRepository".
