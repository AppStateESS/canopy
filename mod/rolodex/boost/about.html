Rolodex

By Verdon Vaillancourt

Rolodex provides a multi-purpose directory, suitable for businesses, clubs and organizations of all sorts.

Rolodex ties into several core mods, including users and demographics. This allows user/member data to be shared by any module also using these core services.

Rolodex can be configured in many different ways, from simple to complex. User-definable fields are also available.
