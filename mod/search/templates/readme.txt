Available tags

** search_box.tpl **

Required
--------------
START_FORM, END_FORM, SEARCH


Optional
--------------
MOD_TITLE - drop down list of available modules to search from - not
used by default

GO - search button. says "Search" on it. Can substitute for image
button or leave out entirely


SEARCH_LABEL - translated word for 'Search'


To get the search box to have default text, please look in
your conf/search/config.php file.
